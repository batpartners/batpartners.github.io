# | 출력(Outputs)

| 이름 | 타입 | 설명 |
| :--- | :--- | :--- |
| Joint Planes | Plane | 선택한 유닛의 캘리브레이션된 축 좌표계(평면). |
| Axis Name | String | Joint Planes와 동일한 트리 구조의 축 이름. |
| Info | String | 선택한 유닛의 축 구성 및 유효성 요약. |
| Warnings | String | 미캘리브·잠금 축, 스캔 실패 등 경고 메시지. |
