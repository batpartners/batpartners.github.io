# 1. PackageManager

Rhino `PackageManager`에서 `GERTY`를 찾아 가장 최신 버전을 설치 후, 라이노를 재실행한다.<br>
Trial License로 자동 실행되며 **30일**간 사용할 수 있다.<br>
`Automatically update packages when Rhino starts` 가 선택 되었다면 최신의 거티를 사용할 수 있다.

	<a href="https://i.postimg.cc/dDMMFGCG/Package-Manager.png"><img src="https://i.postimg.cc/dDMMFGCG/Package-Manager.png"></a>
</figure>



# 2. GetHardwareIDs

Trial License 기간이 만료되었을 경우, GERTY 유료 서비스를 구독할 수 있다.<br>
License 구매 시, GetHardwareIDs 컴포넌트을 꺼내서 고유의 MAC주소 데이터를 출력한다.<br>
출력된 데이터를 `Copy Data Only` 한다. `Copy Data Only`는 패널에서 마우스 오른쪽 클릭시 확인할 수 있다.

<figure>
	<a href="https://i.postimg.cc/g0c21Pm6/Machine-ID.png"><img src="https://i.postimg.cc/g0c21Pm6/Machine-ID.png"></a>
</figure>


복사한 MAC주소는 Contect Support 버튼에 연결된 페이지 양식에 기재 후 제출한다.<br>
만일 해당 양식이 보이지 않을 경우, **contact@b-at.kr** 로 이메일을 보낸다.


# 3. FileImportAndExport

트라이얼 기간이 끝난 이후 이메일을 통해 라이센스를 받을 수 있다.<br>
받은 라이센스는 Grasshopper 상단 리본의 'GERTY Ver.2' > 'Activate GERTY License' 탭 경로에 붙여넣는다.<br>
이후, 라이노 창을 완전히 종료 후 다시 실행하여 설치를 마친다.

<figure>
	<a href="https://i.postimg.cc/ZR7VtN2n/License-Active.png"><img src="https://i.postimg.cc/ZR7VtN2n/License-Active.png"></a>
</figure>

설치가 완료되면 GERTY Ver.2에서 남은 기간과 GERTY를 활용한 템플릿을 실행하여 사용할 수 있다.<br>
이외의 다른 상황에 대해서는 contact@b-at.kr 로 문의할 수 있다.<br>
